### 使えるSVGタグ

| タグ名           | 説明                |
|----------------|---------------------|
| svg            | 一部に制限あり         |
| g              | グループ化するためのコンテナ     |
| path           | パス                  |
| glyph          | グリフ定義             |
| glyphref       | グリフ                 |
| marker         | ハイライト               |
| view           | SVG用               |
| circle         | 円                  |
| line           | 直線                |
| polygon        | 閉じた図形            |
| polyline       | 一連の直線           |
| rect           | 角の丸まった四角形      |
| text           | テキストからなるグラフィクス要素  |
| textpath       | テキストをレンダリング用パス     |
| tref           | 直接埋め込まれた文字データ |
| tspan          | テキストの位置を調整      |
| clippath       | クリッピングパス            |
| filter         | カスタムフィルター           |
| lineargradient | 線形グラデーション         |
| radialgradient | 放射状グラデーション       |
| mask           | アルファマスク             |
| pattern        | 同じ図形の繰り返し描画  |
| vkern          | 垂直距離            |
| hkern          | 水平距離            |
| defs           | グラフィックオブジェクト        |
| use            | 複製                |
| symbol         | グラフィカルテンプレートオブジェクト  |
| desc           | 説明がテキストのみ         |
| title          | 説明がテキストのみの文字列  |
